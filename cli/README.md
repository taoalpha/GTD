## CLI for GTD


### Usage

```
gtd "reading @{everyday} @(8pm - 10pm) #(commute)"
gtd "coding @{MWF} @7:30pm + 2h #(home)"
```

summarize a day:

```
gtd today
# list of all items:
# choose done or fail for each
```

### Symbols

```
`#` / `#()` - place
`@()` - time
`@..+..` - time
`@{}` - frequency

// TDB
`@<>` - 
`#{}` -
```
